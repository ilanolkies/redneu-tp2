## Setup

## Setup

Usar Python: v3.6.5

Para instalar version de Python, la primera vez:

1. Instalar `pyenv` - https://github.com/pyenv/pyenv#installation

  En mac:

  ```
  brew update
  brew install pyenv
  ```

2. Inicializar `pyenv`

  ```
  eval "$(pyenv init -)"
  ```

3. Instalar v3.6.5

  ```
  pyenv install 3.6.5
  ```

4. Activar v3.6.5 en terminal

  ```
  pyenv global 3.6.5
  ```

5. Verificar version

  ```
  python --version
  > Python 3.6.5
  ```

6. Instalar dependencias

  ```
  pip install -r requirements.txt
  ```

Para volver a activarlo en una nueva terminal - setup de primera vez ya terminado

```
eval "$(pyenv init -)"
pyenv global 3.6.5
```
